# Box-Office-Analyzer
Analyze Star Wars movie box office numbers with the programing language R!
A R project demonstrating how to use:
- Vectors  
- Matrices  
- Naming rows/columns  
- Data frames  
- Lists  
- Basic arithmetic and selection  

The project analyzes box office earnings for the six (and only) Star Wars movies.

---

## Features

- Builds a matrix of domestic and international earnings for each film
- Computes worldwide results  
- Converts the data into a data frame  
- Identifies the highest-grossing movie  
- Calculates total franchise revenue  
- Stores all results in a clean list
